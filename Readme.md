#  ChapaCalc – Sistema Inteligente de Conversão de Chapas

##  Sobre o Projeto

O **ChapaCalc** é uma aplicação desenvolvida em Python para resolver um problema real da indústria: converter o peso total em KG informado na nota fiscal para a quantidade real de chapas recebidas.

Na prática industrial, as notas fiscais informam apenas o peso total, enquanto o controle de estoque é feito por unidade. Esse sistema automatiza esse processo, reduz erros manuais e padroniza os cálculos.

---

##  Funcionalidades

- Cadastro de fornecedores e gramaturas
- Seleção de tipo de onda (B ou C)
- Cálculo automático do peso unitário da chapa
- Conversão de KG para quantidade de chapas
- Interface gráfica intuitiva (Tkinter)
- Tratamento de erros e validação de dados
- Versão executável (.exe)
